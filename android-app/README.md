# Stella Android App

This is the Android client for the Stella Voice Assistant.

## Features
- Works when the screen is OFF or locked
- Triggered by Nothing CMF Buds media button
- Records short audio clips (1.6s)
- Sends audio to server (`Config.SERVER_URL`)
- Receives MP3 voice reply from AI
- Plays Stella's voice through earbuds

## Build
APK can be built using GitHub Actions (Step 17).
